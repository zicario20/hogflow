# HogFlow Problem Statement

## Problem category

HogFlow is investigating the general problem category of manual livestock count reconciliation during sequential animal movement through constrained operational passages.

The project does not assume that JBS, Smithfield, Tyson, or any other named company has a proven counting problem.

The project does not claim that count errors are frequent, that financial loss is occurring, that downtime is occurring, or that labor savings have been demonstrated.

HogFlow is investigating whether automated visual counting could provide an independent count estimate within this type of operational context.

## Research context

The concept is motivated by a workflow in which animals may move in groups or sections through a physical process.

Relevant generic context terms include:

* livestock facility
* receiving or processing workflow
* weighing area
* constrained alley
* operator
* section

This description is intentionally generic. It avoids confidential employer-specific operational claims and does not assume that one facility's workflow is universal.

## Current manual-counting concern

The conceptual concern is that a human operator continuously counting moving animals may face a cognitively demanding task, particularly when movement is continuous, animals overlap, or direction changes occur.

This is a research motivation.

It is not presented as a validated failure of any current company process.

## Core research question

Research question:

> Can a computer vision system combining animal detection, multi-object tracking, and directional virtual-line crossing estimate the number of unique pigs moving through a constrained alley with sufficiently low count error to justify further operational evaluation?

## Primary outcome

The primary technical outcome is count accuracy against human-verified ground truth.

HogFlow is evaluated as a counting system.

Detection accuracy alone is not the final project outcome.

## Non-goals

Phase 0 does not prove:

* production readiness
* operational viability
* validated accuracy
* reduced labor
* reduced downtime
* financial savings
* commercial demand
* compatibility with every pork-processing facility
